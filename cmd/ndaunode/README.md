# `ndaunode`: ABCI application implementing a ndau chain node

## Usage

- Terminal 1:

    ```sh
    $ ndaunode
    ```

- Terminal 2:

    ```sh
    $ tendermint node
    ```

- Terminal 3:

    Set and read values from the ndau blockchain using the `ndau` tool:

    ```sh
    $ ndau set -k key -v value
    ```

    and so forth. See `ndau` tool documentation for more details.
